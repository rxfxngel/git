# GIT
##  Comandos para bitbucket
### Ver cambios en el repositorio
```git
git status
```
### Agregar cambios
```git
git add datos.txt
```
### Confirmar cambios
```git
 git commit -m 'nuevo txt'
```
### Agregar Origen
```git
git remote add origin https://github.com/rxfxngel/safepass.git
```
### publicar cambios en la nube
```git
git push -u origin master
```
